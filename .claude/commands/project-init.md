# /project-init — Project Context Initializer

Gunakan command ini saat pertama kali masuk ke project baru, atau saat `.claude/PROJECT_CONTEXT.md` belum ada.

## Instruksi untuk Claude Code

Ketika user menjalankan `/project-init`, lakukan langkah-langkah berikut **secara otomatis tanpa nanya berulang**:

### Step 1: Auto-detect project
Jalankan perintah berikut untuk scan project:
```bash
# Detect tech stack
ls package.json composer.json requirements.txt Cargo.toml pubspec.yaml 2>/dev/null
cat package.json 2>/dev/null | head -30
cat pubspec.yaml 2>/dev/null | head -30

# Detect framework
grep -r "flutter\|react\|laravel\|django\|express\|nextjs" package.json pubspec.yaml composer.json 2>/dev/null | head -5

# Detect structure
ls -la
find . -name "*.md" -maxdepth 2 2>/dev/null
```

### Step 2: Read existing docs
Baca semua file dokumentasi yang ada:
- README.md
- docs/ folder
- Existing .claude/ files

### Step 3: Generate PROJECT_CONTEXT.md

Buat file `.claude/PROJECT_CONTEXT.md` dengan template ini, diisi berdasarkan hasil scan:

```markdown
# PROJECT_CONTEXT.md
> Auto-generated by /project-init — Edit sesuai kebutuhan

## 📦 Project Info
- **Nama:** [detected dari package.json/pubspec.yaml]
- **Stack:** [detected stack]
- **Framework:** [detected framework]
- **Database:** [detected jika ada]

## 🏗️ Arsitektur
[Tulis ringkasan arsitektur berdasarkan struktur folder]

## 📁 Folder Penting
[List folder kunci dan fungsinya]

## ⚙️ Setup & Run
```bash
# Install
[detected install command]

# Run dev
[detected run command]

# Test
[detected test command]
```

## 🎯 Fitur Utama
[List fitur yang ada berdasarkan scan kode]

## 🔄 Konvensi Kode
- [Detected naming convention]
- [Detected code style]

## 📌 Status Saat Ini
- Branch aktif: [git branch]
- Last commit: [git log --oneline -1]
- Open issues: [jika ada TODO/FIXME di kode]

## 🚀 Next Steps
[Tulis berdasarkan README atau context yang ada]
```

### Step 4: Create log file
```bash
mkdir -p .claude/logs
echo "# Claude Action Log — $(date)" > .claude/logs/action-log.md
echo "" >> .claude/logs/action-log.md
echo "[$(date -Iseconds)] INIT: Project context initialized via /project-init" >> .claude/logs/action-log.md
```

### Step 5: Report
Setelah selesai, laporkan:
- Apa yang terdeteksi
- File yang dibuat
- Skill yang relevan untuk project ini
- Command yang berguna untuk project ini
